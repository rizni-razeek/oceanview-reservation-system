Help Section

This page provides simple guidance for staff members using the reservation system.

1. Login

Enter your username and password on the login page to access the system.

2. Add New Reservation

Use the “Add New Reservation” option from the dashboard. Fill in all required guest details, room type, and stay dates, then click “Save Reservation”.

3. View Reservation Details

Use the “View Reservation Details” page and enter a reservation number to see the full booking information.

4. View All Reservations

Select “View All Reservations” to see the full reservation list. You can also search by reservation number or guest name.

5. Edit Reservation

From the reservations table, click the “Edit” link next to a booking. Update the details and submit the form.

6. Delete Reservation

From the reservations table, click the “Delete” link next to a booking to remove it from the system.

7. Calculate and Print Bill

Open the bill page, enter the reservation number, and generate the bill. Use the print button to print the final bill.

8. Logout / Exit

Use the “Logout” link from the dashboard to safely end your session before leaving the system.
